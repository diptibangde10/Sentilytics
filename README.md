# Sentilytics

**Sentilytics** is a modern web app for **analyzing product review sentiments** using an interactive and visually appealing interface.
Built with **React**, **TypeScript**, and **Tailwind CSS**, it helps businesses and researchers **visualize customer feedback** efficiently and gain actionable insights.

---

## 🚀 Features

* 📊 Analyze product reviews to identify **positive, negative, and neutral sentiments**.
* ⚡ **Fast and smooth performance** powered by Vite and React.
* 📱 Fully **responsive UI** for desktop and mobile users.
* 🎨 Clean design with **Tailwind CSS** and **shadcn/ui components**.

---

## 🛠 Technologies Used

* **React** – UI library
* **TypeScript** – Strongly typed JavaScript
* **Vite** – Fast build tool and development server
* **Tailwind CSS** – Utility-first CSS framework
* **shadcn/ui** – Accessible headless UI components

## ✅ Requirements

Before starting, make sure you have the following installed:

1. **Node.js (v18 or higher)**

   ```bash
   node -v
   ```

   Should return a version like `v18.x.x`. Install via [Node.js official site](https://nodejs.org/) or use **nvm**:

   ```bash
   nvm install 18
   nvm use 18
   ```

2. **npm** (comes with Node.js)

   ```bash
   npm -v
   ```

3. **Visual Studio Code (VS Code)** or another preferred editor.

4. **Git**

   ```bash
   git --version
   ```

---

## 📥 Installation

1. **Clone this repository:**

```bash
git clone https://github.com/diptibangde10/sentilytics.git
cd sentilytics
```

2. **Install dependencies:**

```bash
npm install
```

3. **Run the development server:**

```bash
npm run dev
```

After running this command, you’ll see a local URL (e.g., `http://localhost:5173`). Open it in your browser to explore the app.

---

## ▶️ Usage

* **Upload product reviews** or connect to a review dataset.
* View **real-time sentiment breakdowns** with charts and insights.
* Filter reviews based on sentiment type and product category.

## ⚙️ How It Works

1. The app loads customer review data into an **interactive dashboard**.
2. Each review is processed and classified as **positive**, **negative**, or **neutral**.
3. Results are visualized using dynamic charts for **better understanding of customer sentiment**.
